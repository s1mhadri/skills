# skills
List of skills I use personally with coding agents.

## Quickstart

1. Run the skills installer
```sh
npx skills@latest add s1mhadri/skills
```

2. Pick the skills you want, and which coding agents you want to install them on.

3. All set. Ready to use.

## Available skills

### 1. Project Implementation Teacher

Makes the Agent or LLM teach you by implementing a project or a concept step by step.

The main problem I faced while learning a topic using Agent/LLM was that it gave lot of information at a time. Kind of like information dump. This skill slows down the Agent/LLM to go one step at a time and let's you clear your doubts before moving ahead.

It also creates a learning notes progressively so that you can look back in the future.

**When to use this**

- Learn a concept by implementing a project.
- Implement a new feature in an existing project
- Learn any type of coding related concepts
